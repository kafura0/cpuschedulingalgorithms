1.Run the program in a ide that supports java
2.A prompt requesting for number of process will appear. Key in the number of process and press enter.
3.A prompt requesting for burst time will appear. Key in the first number and press enter. Followed by the second burst time. Repeat until
each process has a burst time.
3.A prompt requesting for priority will appear. Key in the first number and press enter. Followed by the second priority. Repeat until
each process has a priority.
4.Wait for the results.
